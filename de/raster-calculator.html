Verwendung des Rasterrechners¶

  1. Wählen sie Datei ‣ Neues Projekt

  2. Klicken sie auf Layer ‣ Rasterlayer hinzufügen...

  3. Öffnen sie die Datei /home/user/data/raster/SRTM_u03_n041e002.tif

  4. Klicken sie auf Raster ‣ Rasterrechner...

  5. Geben sie eine Formel und eine Datei für das Resultat ein:

  6. Doppelklicken sie mountains Im Layerbaum und setzen sie :guilabel: Einkanaleigenschaften –>Farbabbildung auf Pseudofarben

  7. Kombinieren sie beide Raster, um alle Punkte unter 1000 Meter zu löschen:

  8. Das Resultat ist ein DEM ohne Punkte unter 1000 Meter:
