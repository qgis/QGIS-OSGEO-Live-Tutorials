Using the raster calculator¶

  1. Choose File ‣ New Project

  2. Click Layer ‣ Add Raster Layer...

  3. Open /home/user/data/raster/SRTM_u03_n041e002.tif

  4. Click Raster ‣ Raster calculator...

  5. Enter output raster file and a formula:

  6. Double click mountains in the Layers tree and set the color map style to Pseudocolor

  7. Combine both rasters to remove all points lower than 1000 meters:

  8. The resulting raster is a DEM without the points lower than 1000 meters:
