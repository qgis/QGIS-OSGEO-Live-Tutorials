Labelling of city layer
================================================================================

#. Choose :menuselection:`File --> Open Project...` from the menu bar

#. Select :file:`QGIS-NaturalEarth-Example.qgs` and press :guilabel:`Open`

#. Check :file:`10m_populated_places_simple` in the Layers tree

#. Click labelling tool (ABC) in the toolbar

#. Activate layer labelling with the following settings and press :guilabel:`Apply`:

   .. image:: ../images/label_settings.png
     :scale: 70 %

#. Click on tab :guilabel:`Data defined settings`, set a data defined font size and press :guilabel:`Apply`:

   .. image:: ../images/label_data_defined_settings.png
     :scale: 70 %


.. Data defined label position in labeling-ng http://linfiniti.com/2011/03/interactive-label-placement-in-qgis-1-7/

#. Activate diagram display with the following settings and press :guilabel:`Apply`:
